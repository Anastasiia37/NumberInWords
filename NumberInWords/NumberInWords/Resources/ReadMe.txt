This application converts an integer into a number in words. 
The application starts from the command line.

Input parameters:  <number>  [<language>]
WHERE:
<number> is unsigned long integer (in range from 0 to 999.999.999.999 for russian and from 0 to 999.999 for english),
<language> is ru (for russian) or eng (for english). 
If no language is specified, the default language (russian) is used.
